# Portfólio de Douglas Sales

Bem-vindo ao meu portfólio! Este projeto é uma vitrine das minhas habilidades em **desenvolvimento web e back-end**, apresentando meus projetos, certificados e informações de contato.

## Sobre o Projeto

O portfólio foi desenvolvido para mostrar minhas especialidades e projetos de forma clara e moderna, utilizando tecnologias atuais para garantir responsividade e boa experiência do usuário.

Principais seções:
- **Início:** Apresentação com foto e breve descrição sobre mim.
- **Especialidades:** Minhas competências em desenvolvimento web, back-end e bancos de dados.
- **Sobre:** Detalhes sobre minha formação, experiência e links para redes sociais.
- **Projetos:** Galeria dos meus projetos com links para GitHub.
- **Certificados:** Visualização de cursos e certificações.
- **Contato:** Formulário para enviar mensagens diretamente para mim.
- **Currículo:** Download do meu currículo atualizado.

## Tecnologias Utilizadas

- **HTML5** – Estrutura semântica do site.
- **CSS3** – Estilização personalizada, animações e responsividade.
- **Bootstrap Icons** – Ícones modernos e interativos.
- **Formspree** – Para envio de formulários de contato sem back-end.
- **Fonts Google (Poppins)** – Tipografia elegante e legível.

## Funcionalidades

- Layout responsivo, compatível com desktop e mobile.
- Efeitos de hover em botões, imagens e seções de destaque.
- Seção de projetos com overlay para links e descrição.
- Integração de redes sociais (LinkedIn, GitHub, Instagram).
- Formulário de contato funcional via Formspree.
- Download de currículo em PDF.
